# Creative

## Objetivo

Detectar fatiga creativa y oportunidades de mejora en anuncios.

## Problema que se resolvió

El equipo no tenía una forma rápida de identificar cuándo una creatividad debía iterarse, escalarse o retirarse.

## Qué se hizo

- Módulo de diagnóstico creativo.
- Señales de rendimiento por anuncio.
- Lectura integrada con costo y resultados.

## Cómo se resolvió

Se consolidaron indicadores operativos para clasificar creativos en tres estados: escalar, mantener o reemplazar.

## Decisiones que habilita

- Iterar copys y piezas con baja respuesta.
- Escalar variaciones ganadoras.
- Reducir desperdicio en anuncios fatigados.

## Nota de gestión

La creatividad se evalúa junto con audiencia y puja; no se recomienda intervenir de forma aislada.
