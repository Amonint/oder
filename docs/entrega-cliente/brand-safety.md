# Brand Safety

## Objetivo
Reducir riesgo reputacional y operativo en la ejecución de pauta.

## Problema que se resolvió
Faltaba un marco explícito para validar que decisiones de rendimiento no afecten la percepción de marca.

## Qué se hizo
- Se incorporó criterio de revisión de mensajes y públicos.
- Se documentaron controles mínimos de operación.
- Se alineó el análisis de performance con lineamientos de marca.

## Cómo se resolvió
Se definió una práctica de doble validación:
1. Validación de rendimiento (datos).
2. Validación de alineación de marca (mensaje, contexto, audiencia).

## Checklist mínimo
- El copy respeta tono y propuesta de valor.
- No hay promesas ambiguas o riesgosas.
- Segmentación consistente con público objetivo.
