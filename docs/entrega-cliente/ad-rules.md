# Ad Rules

## Objetivo
Estandarizar reglas de decisión para acelerar la operación sin perder control.

## Problema que se resolvió
Las decisiones de pausar/escalar anuncios podían variar según el analista, generando inconsistencias.

## Qué se hizo
- Reglas base para clasificar anuncios por desempeño.
- Criterios de intervención por tendencia.
- Marco de priorización de acciones.

## Cómo se resolvió
Se aplicó un enfoque por estados:
- **Escalar**: mejora consistente y costo controlado.
- **Monitorear**: señales mixtas sin evidencia suficiente.
- **Intervenir**: deterioro sostenido en eficiencia o calidad.

## Recomendación
Registrar cada acción aplicada para medir impacto en la siguiente ventana de análisis.
